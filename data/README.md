# data

This directory stores experiment data used by the reproducibility scripts.

## Subdirectories

- `simulated/`: simulated linear Gaussian Bayesian-network data generated by the experiment scripts.
- `benchmark/`: optional location for external benchmark networks such as Asia, Sachs or ALARM. The main pipeline does not depend on this directory.

## File Types

- `*_data.csv`: simulated observation matrix.
- `*_adj.csv`: ground-truth DAG adjacency matrix.
- `*_beta.csv`: coefficient matrix for the linear structural equations.

The main scripts reuse existing files in `simulated/` when available and generate missing files when necessary.
